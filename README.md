# ScrumPokerFire

Scrum poker for your plannings.

## Scripts

``` bash
# Install dependencies
$ npm install

# Start application
$ npm start

# Build
npm run build
npm run build:prod

# Lint
$ npm run lint
$ npm run lint:fix

# Test
$ npm run test

# Deploy to Firebase
npm run deploy

# Clean
$ npm run clean                   # Removes dist and .angular folders
$ npm run clean:packages          # Removes node_modules folder
$ npm run clean:all               # Runs all of the above
```
